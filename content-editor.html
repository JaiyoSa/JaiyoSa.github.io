Generate blog posts
without writing code.

Fill the form, download the Markdown file, then place it inside the posts folder. Add the filename to posts/posts.json.

New blog post

After downloading

  1. Move the .md file into the posts/ folder
  2. Open posts/posts.json in a text editor
  3. Add the filename (e.g. "2026-05-01-my-post.md") to the list
  4. Push to GitHub with:
    git add -A && git commit -m "New post" && git push
